# Java Market Simulation Project
## 🚀 Features
- Product selection and real-time total price calculation.
- Discount coupon system (using the code: DISCOUNT10).
- Payment processing and change (balance) calculation logic.

### 🛠 Core Concepts Used
- `Scanner`: For handling user input.
- `While Loop`: For managing the continuous shopping cycle.
- `Switch-Case`: For product selection and menu navigation.
- `If-Else`: For logical decision-making, coupon validation, and payment checks.
